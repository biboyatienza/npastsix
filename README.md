## Description

Nth past 6 project is the effect of the 24 hour API challenge given on Sun, 22 May, 06:00 using Nest. Now let's try to build something usefull base on it.    


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
